[모바일 반응형 UI 업데이트 반영내역]

1. 모바일(768px 이하)에서 좌측 사이드바 숨김 처리
2. 모바일 상단 고정 헤더 추가
   - 좌측 로고/타이틀
   - 우측 햄버거 메뉴(☰)
3. 햄버거 메뉴 클릭 시 슬라이드 메뉴 패널 오픈
4. 메뉴 선택 시 패널 자동 닫힘
5. 배경 클릭 또는 X 버튼으로 메뉴 닫기 가능
6. 모바일 콘텐츠 영역 전체 너비 사용
7. PC에서는 기존 좌측 사이드바 구조 유지

[변경 파일]
- src/components/Layout.tsx

[적용방법]
1. ZIP 압축 해제
2. 기존 프로젝트 폴더에 덮어쓰기
3. dataService.ts 안의 Apps Script URL 유지 여부 확인
4. npm install
5. npm run dev
6. 모바일에서 확인

[Vercel 반영]
1. GitHub에 수정 파일 업로드/커밋
2. Vercel 자동 재배포 확인
