# 창호마스터 이음 광고 대시보드 - 자동 DB 판별 버전

## 이번 버전 핵심
- DB 업로드 버튼은 하나만 사용
- 관리시스템 원본 엑셀 그대로 업로드
- 컨설팅UTM 파일은 자동으로 1차 원본으로 판별
  - 주소/건물명/견적 params 있음 → 1차 DB(C퍼널)
  - 이름/연락처만 있음 → 리타겟 DB
- 컨설팅리스트 파일은 자동으로 2차 DB(상담신청)로 판별
- 2차 DB의 유입경로가 홈페이지/direct인 경우, 같은 연락처의 1차 DB UTM 매체를 자동 승계
- 직접유입(direct), 기타(etc) 채널 추가

## 실행
npm install
npm run dev

## 권장 업로드 순서
1. 1차DB 파일(컨설팅UTM) 먼저 업로드
2. 2차DB 파일(컨설팅리스트) 업로드

이 순서로 해야 2차DB가 1차DB의 UTM 매체를 정확히 승계합니다.
